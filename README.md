## How to run this program

1. Create a new venv

```sh
python -m venv .venv
```

Note: This code was built in Python: 3.10

Windows:

```sh
source .venv/Scripts/activate
```

Linux:

```sh
source .venv/bin/activate
```

2. Install dependencies

```sh
pip install -r requirements.txt
```
